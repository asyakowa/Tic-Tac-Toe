# 🎮 Tic Tac Toe (Крестики-нолики)

Android-приложение классической игры "Крестики-нолики", разработанное на Kotlin.

---

## 📱 Описание проекта

Пользователь играет против ИИ, пытаясь первым собрать линию из 3 символов
(по горизонтали, вертикали или диагонали).

Проект разрабатывается в рамках обучения Android-разработке.

---

## 🚧 Статус проекта

Проект находится в процессе разработки.  
На текущем этапе реализуется пользовательский интерфейс.

---

## 📱 Экраны

Планируемые экраны приложения:

- Главное меню
- Экран настроек
- Игровой экран

### 🔧 Текущий прогресс

В разработке: экран главного меню
